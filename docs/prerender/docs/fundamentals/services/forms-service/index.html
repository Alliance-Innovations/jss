Forms Service

To enable consuming Sitecore Forms from JSS, APIs are provided to:

  • Get form data from Layout Service using a Rendering Contents Resolver
  • Accept form posts and return JSON data in a REST service. The URL for for this API is /api/jss/formbuilder

The usage and setup of the forms service is detailed in the forms reference documentation.
